# Motion Detection in Live Video

This Python script utilizes OpenCV to detect motion in a live video stream. The program captures frames from the default camera and compares the intensity of the blue channel at a specific pixel location (100,100) between the current frame and a reference image. If a change in intensity is detected, it increments a counter and displays a message indicating potential motion.

## Prerequisites
- Python
- OpenCV library
